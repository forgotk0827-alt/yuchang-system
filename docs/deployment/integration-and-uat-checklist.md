# 联调与真机验收清单

## 本地联调准备

1. 启动后端服务

```bash
npm run dev
```

2. 使用微信开发者工具打开项目根目录

3. 确认小程序请求地址为：

```text
http://127.0.0.1:3000
```

4. 开发者工具已关闭 URL 校验

## 账号联调

- 普通员工：`E001 / E001@`
- 部门评估组：`D001 / D001@`
- 财务复核：`F001 / F001@`
- 精益办复审：`L001 / L001@`
- 超级管理员：`A001 / A001@`

## 业务联调顺序

### 1. 提案闭环

- 普通员工登录
- 新建草稿
- 进入草稿再次编辑
- 上传附件
- 提交提案
- 部门评估组通过
- 财务复核或精益办复审
- 评审委员会核准
- 查看积分到账

### 2. 兑换闭环

- 员工进入兑换页
- 提交礼品兑换
- 精益办或管理员审核
- 发放登记
- 员工查看兑换结果

### 3. 管理闭环

- 管理中心查看员工列表
- 上传员工 Excel
- 查看报表中心
- 查看操作日志
- 执行年度清零接口验证

### 4. 账号安全

- 用户修改密码
- 退出登录
- 使用新密码重新登录

## 真机验收重点

- 登录不再出现“网络请求失败”
- 草稿可以重新进入编辑
- 只有草稿允许删除
- 待办审批角色展示正确
- 移动端按钮和文本不重叠
- 上传附件成功
- 礼品库存和积分扣减同步正确
- 操作日志记录完整

## 正式上线前最终确认

- 客户已提供正式域名和 SSL
- 微信小程序后台已配置合法域名
- 已完成备案
- 已确认员工初始密码规则
- 已确认评审委员会名单
- 已确认积分清零时间口径
- 已确认礼品目录版本和库存数量
