# 煜昌电器改善提案系统

本项目是煜昌电器改善提案、积分激励、礼品兑换与统计报表的一体化 Web 系统原型。

## 功能范围

- 登录与角色权限：普通员工、部门初审、财务复核、精益办复审、超级管理员
- 改善提案：草稿、提交、部门初审、财务复核、精益办复审、驳回、归档
- 附件管理：提案附件与财务效益核算表上传、下载
- 积分账户：积分余额、流水、人工奖励、人工扣减
- 兑换管理：礼品库存、库存预占、兑换审核、发放确认
- 数据导入：员工、部门、礼品 CSV / XLS / XLSX 模板下载与导入
- 报表与日志：提案统计、积分排行、操作日志

## 本地运行

```bash
npm run dev
```

访问：

```text
http://localhost:3000
```

## 数据说明

- 本地运行数据保存在 `data/db.json`，该文件不会提交到 Git。
- 上传附件保存在 `data/uploads/`，该目录不会提交到 Git。
- 如需重置演示数据，停止服务后删除 `data/db.json`，再重新启动即可。
- 员工导入兼容字段：`姓名`、`工号`、`1级部门`、`2级部门`、`部门主管`、`手机号`、`职位`、`员工状态`、`性别`。

## 技术说明

当前版本为零依赖 Node.js 实现，便于快速演示和确认业务流程。后续正式部署可替换为 PostgreSQL/MySQL、对象存储和更完整的鉴权方案。
